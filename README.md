# Rat Cash Deniability Solution

Rat Cash is a simple fork of [Tornado Cash](https://etherscan.io/address/0x910cbd523d972eb0a6f4cae4618ad62622b39dbf) that gives users financial privacy without being associated with criminal activity.

At withdrawal, the user specifies a blacklist of deposits and proves that it is not withdrawing any of them, thus "ratting out" depositors on the blacklist.
It was inspired by @Agusx1211's [tweet](https://twitter.com/Agusx1211/status/1556735814515785728).
